1. Create Project Directory
mkdir money-tracker
cd money-tracker

2. Initialize Node.js Project
npm init -y


3. Install Dependencies
npm install express mongoose body-parser


4. File Structure
money-tracker/
├── public/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── scripts.js
│   └── index.html
├── models/
│   └── transaction.js
├── routes/
│   └── transactions.js
├── server.js
└── package.json

5 Start the server
node server.js
